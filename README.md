# Python Practice Projects 

Welcome to my Python practice repository! This folder is dedicated to building a solid foundation in Python, starting from core concepts like variables, data types, and loops, and moving towards advanced programming logic.

## 📂 Project List & Descriptions

Here are the programs I am currently working on to practice my Python skills:

### 1. Student Enrollment Record (Data Analyst Course)
* **Description:** A program to manage and print student enrollment details. It practices taking user inputs, validating strings, and organizing different data types (Strings, Integers, Booleans) into a structured format.
* **Key Concepts Used:** Variables initialization, User Inputs, Type Casting, `for` loops, Conditional statements (`if/else`, `break`).

### 2. Circle Stats Calculator
* **Description:** A math-based program that takes the radius of a circle as input and automatically calculates its Area and Circumference.
* **Key Concepts Used:** Constants, Arithmetic Operators, Math formulas integration.

### 3. Find the Middle Number
* **Description:** A logical program that takes three different numbers as input and uses comparison logic to filter out the exact middle number.
* **Key Concepts Used:** Multiple variable comparisons, Logical operators (`and`, `or`), Nested conditional logic.

### 4. Digit Extract Game
* **Description:** A clever math-based puzzle/game that takes a multi-digit number (e.g., a 3-digit number like 123) and breaks it down to extract each individual digit into its own variable.
* **Key Concepts Used:** Modulo operator (`%`), Floor division (`//`), Variable re-assignment.
